# Установка Google Apps Script
1. Откройте Google Таблицу.
2. Меню `Расширения` → `Apps Script`.
3. Замените содержимое скрипта на код из файла `apps_script.gs`.
4. Сохраните и опубликуйте: `Развернуть` → `Веб-приложение`:
   - Новый проект
   - Доступ: **Кому угодно**
   - Метод: **POST**
5. Вставьте полученный URL в `script.js` вместо `YOUR_GOOGLE_SCRIPT_URL`.
